3.1 token identification for keywords in your lexical analyzer.

Detect these exact keywords in source code:
if, else, for, while, do, break, int, char, float, double, void, return, switch, case, default, continue, goto, printf
When found, print the corresponding token in output as <TOKEN> form.
Examples:
if → <IF>
else → <ELSE>
printf → <PRINTF>
